# Security Policy

## Supported Versions

Currently, we are focusing our security efforts on our first stable release.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you find a security vulnerability, please **do not** open a public issue. We want to keep XDB-Project safe for everyone.

### How to report:
Please report any security issues by sending an email to:
**xdbproject@gmail.com**

Please include:
*   A description of the vulnerability.
*   The steps to reproduce it.
*   Your GitHub username if you wish to be credited in our security advisories.

We will respond within 48 hours to acknowledge your report and discuss the next steps.
